# 📊 Chart.js Data Visualization Demo

A simple and clean web project demonstrating how to use the Chart.js library to create interactive charts using HTML, CSS, and JavaScript.

This project contains:

- 📈 Line Chart — Monthly sales visualization
- 🥧 Pie Chart — Budget distribution visualization
- 🎨 Modern responsive UI design

---

## 🚀 Features

- Responsive layout
- Clean modern styling
- Interactive charts using Chart.js
- Easy to customize
- Beginner-friendly structure

---

## 📂 Project Structure

```bash
project-folder/
│
├── index.html
├── style.css
├── script.js
└── README.md
````

---

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript
* Chart.js

---

## 📋 How to Run

1. Download or clone the project.
2. Make sure all files are in the same folder:

   * `index.html`
   * `style.css`
   * `script.js`
3. Open `index.html` in your browser.

---

## 📈 Charts Included

### 1. Line Chart

Displays monthly sales data using a smooth line graph.

### 2. Pie Chart

Shows budget distribution categories in a circular chart.

---

## 🔗 Chart.js CDN

The project uses Chart.js from CDN:

```html
<script src="https://cdn.jsdelivr.net/npm/chart.js@4.4.2/dist/chart.umd.min.js"></script>
```
